  • 기존 외주 개발 시스템 마이그레이션 작업
    • 기존 PHP CodeIgniter + 카페24 호스팅 환경을 → Ruby on Rails + AWS 기반으로 마이그레이션
    • 최적화되지 않은 MySQL 데이터베이스에 대해 인덱스 추가, 비정규화된 테이블 구조 정리 및 마이그레이션 수행
    • 외주로 제작된 iOS 앱의 기술적 한계를 극복하기 위해 Swift로 신규 앱 개발
    • 예약, 결제, 정산 도메인 전반에 대한 구조 설계 및 개선
  • Ruby on Rails 기반 웹 서버 애플리케이션 개발 및 운영
    • RESTful API 설계 및 구현
    • 실시간 예약 및 빌링 기능 구현, 결제까지 가능한 단일 예약 플로우 구성
    • 관리자용 결제 정산 백오피스를 구축하여 운영 효율성 향상
    • 지도 데이터 최적화 쿼리 구현
    • Queue와 Worker(Sidekiq) 서버를 이용한 대규모 푸시 알림 전송 시스템 구축
  • Swift 기반 iOS 애플리케이션 개발 및 운영
    • MVVM 아키텍처를 도입하여 View의 추상화 달성
    • iOS 앱에 RxSwift를 도입하여 구조 개선 및 비동기 흐름 안정화
    • Storyboard 기반 UI 구축
